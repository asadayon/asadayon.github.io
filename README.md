- Featured Projects
- Research & Publications
- Contact

## Deployment
This website is designed for **GitHub Pages**.

### Steps
1. Create a repository named `asadayon.github.io`
2. Upload all files
3. Go to **Settings → Pages**
4. Under **Build and deployment**, select **Deploy from a branch**
5. Select **main** branch and **/root** folder
6. Save

Your site should appear at:
`https://asadayon.github.io/`

## Customize
- Replace `assets/profile.jpg` with your headshot
- Replace `assets/cv.pdf` with your CV
- Replace project screenshots inside `assets/`
- Update text inside `index.html` as needed
